## 3. 页面详细说明

### 3.1 首页（对话页面）

#### 3.1.1 页面概述

- **页面路径**：`/`
- **页面组件**：`Home.vue`
- **功能描述**：提供AI对话交互界面，用户可以通过自然语言提问获取游戏相关信息

#### 3.1.2 页面布局

```
┌─────────────────────────────────┐
│         顶部导航栏                │
├─────────────────────────────────┤
│                                 │
│         对话消息区域              │
│     （可滚动，显示历史消息）        │
│                                 │
├─────────────────────────────────┤
│      快捷问题入口（可选）          │
├─────────────────────────────────┤
│      对话输入框 + 发送按钮         │
└─────────────────────────────────┘
```

#### 3.1.3 主要组件

**ChatInput组件**
- 功能：用户输入问题并发送
- 属性：
  - `disabled`：是否禁用输入
  - `placeholder`：输入框占位符
- 事件：
  - `send`：发送消息事件

#### 3.1.4 状态管理

使用`chatStore`管理对话状态：

```javascript
{
  messages: [],           // 消息列表
  isTyping: false,        // 是否正在输入
  currentChatId: null,    // 当前对话ID
  chatHistory: []         // 对话历史
}
```

#### 3.1.5 核心功能

1. **发送消息**
   - 用户输入问题
   - 调用`/api/v1/chat/send`接口
   - 流式接收AI回复
   - 实时展示回复内容

2. **显示历史对话**
   - 从数据库加载历史对话
   - 支持查看历史对话记录

3. **快捷问题**
   - 提供常用问题的快捷入口
   - 点击快捷问题自动发送

#### 3.1.6 API接口

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/v1/chat/send | POST | 发送消息 |
| /api/v1/chat/history | GET | 获取历史对话 |

#### 3.1.7 样式说明

- 采用Flex布局实现上下结构
- 消息区域支持滚动
- 输入框固定在底部
- 响应式设计，适配移动端

### 3.2 英雄列表页

#### 3.2.1 页面概述

- **页面路径**：`/hero`
- **页面组件**：`Hero.vue`
- **功能描述**：展示王者荣耀所有英雄列表，支持搜索和筛选

#### 3.2.2 页面布局

```
┌─────────────────────────────────┐
│         顶部导航栏                │
├─────────────────────────────────┤
│      搜索框                       │
├─────────────────────────────────┤
│      筛选条件（位置/难度）          │
├─────────────────────────────────┤
│                                 │
│      英雄列表（卡片形式）          │
│    （Grid布局，可滚动）            │
│                                 │
└─────────────────────────────────┘
```

#### 3.2.3 主要组件

**HeroCard组件**
- 功能：展示单个英雄信息
- 属性：
  - `hero`：英雄数据对象
- 事件：
  - `click`：点击事件

#### 3.2.4 状态管理

使用`heroStore`管理英雄数据：

```javascript
{
  heroes: [],           // 英雄列表
  loading: false,       // 加载状态
  filters: {            // 筛选条件
    position: '',
    difficulty: '',
    keyword: ''
  }
}
```

#### 3.2.5 核心功能

1. **加载英雄列表**
   - 调用`/api/v1/heroes`接口
   - 支持分页加载

2. **搜索功能**
   - 支持按英雄名称搜索
   - 实时搜索结果展示

3. **筛选功能**
   - 按位置筛选：对抗路/中路/发育路/打野/游走
   - 按难度筛选：简单/普通/困难

4. **英雄排序**
   - 支持按胜率/禁用率/选用率排序

#### 3.2.6 API接口

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/v1/heroes | GET | 获取英雄列表 |
| /api/v1/heroes/:id | GET | 获取英雄详情 |

#### 3.2.7 样式说明

- 采用Grid布局展示英雄卡片
- 卡片悬停效果
- 响应式布局，移动端单列，PC端多列
